# Glide

Glide is a toolkit for building wallet-enabled web games for the Godot game engine.

Current status: Slice 5.2 in progress: the standalone web-shell dependency boundary now uses Phantom's current browser SDK package, on top of the completed Slice 1 to Slice 4 foundation.

Next checkpoint: Slice 5.3 create the provider wrapper boundary around the Phantom browser SDK.
